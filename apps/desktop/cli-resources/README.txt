Forge CLI package

This package installs the terminal-first Forge binaries:

- forge.exe
- forged.exe

Install:
  powershell -ExecutionPolicy Bypass -File .\install-cli.ps1

Uninstall:
  powershell -ExecutionPolicy Bypass -File .\uninstall-cli.ps1

Installed location:
  %LOCALAPPDATA%\Programs\Forge\bin

User data remains under:
  %USERPROFILE%\.forge
